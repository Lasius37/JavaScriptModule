    Créer un tableau

    Un tableau (array) permet de stocker plusieurs valeurs dans une seule variable, en conservant un ordre.

    • Créer un tableau vide : 
      const notes = [];
    • Créer un tableau avec des valeurs : 
      const fruits = ["pomme", "banane", "orange"];
const nombres = [3, 7, 12];
    • Tableau “mixte” (déconseillé) : 
      const data = ["Alice", 12, true];

    Lire un tableau

    Les éléments d’un tableau sont repérés par un index (position), qui commence toujours à 0.

    • Accéder à un élément : 
      const fruits = ["pomme", "banane", "orange"];

console.log(fruits[0]);  // "pomme"
console.log(fruits[2]);  // "orange"
    • Connaître la taille (nombre d’éléments) : 
      console.log(fruits.length);  // 3
    • Accéder au dernier élément : 
      const dernier = fruits[fruits.length - 1];
console.log(dernier);  // "orange"

    Si l'index qui n’existe pas, JavaScript renvoie undefined.

    Modifier un tableau

    On peut modifier un tableau en changeant un élément, en ajoutant, ou en supprimant des éléments.

    • Modifier une valeur (écriture par index) : 
      const fruits = ["pomme", "banane", "orange"];
fruits[1] = "fraise";

console.log(fruits);  // "banane" est remplacé par "fraise"
    • Ajouter à la fin : push() 
      const notes = [10, 12];
notes.push(15);

console.log(notes);  // [10, 12, 15]
    • Retirer le dernier élément : pop() 
      const dernier = notes.pop();
console.log(dernier);  // 15
console.log(notes);  // [10, 12]
    • Ajouter au début : unshift() 
      const file = ["B", "C"];
file.unshift("A");  // ajoute "A" au début
    • Retirer au début : shift() 
      const file = ["A", "B", "C"];
const premier = file.shift();  // retire "A"

    Remarque : même si un tableau est déclaré avec const, on peut modifier son contenu. Ce que const empêche, c’est de réassigner la variable à un autre tableau.

    Parcourir un tableau

    Parcourir un tableau consiste à accéder successivement à chacun de ses éléments. On utilise cela pour afficher des valeurs, faire des calculs ou appliquer une condition sur chaque élément.

    • Boucle classique (si besoin des index) : 
      const notes = [8, 12, 15];

for (let i = 0; i < notes.length; i++) {
    console.log(notes[i]);
}
    • Boucle for...of (si pas besoin des index) : 
      for (const note of notes) {
    console.log(note);
}

    Erreurs fréquentes

    • Oublier que les index commencent à 0 : comportement inattendu.
    • Confondre length et dernier index : comportement inattendu.
    • Lire un index inexistant et ne pas gérer undefined : risque d'erreurs multiples.
    • Confondre for...in et for...of : parcours incorrect du tableau.
