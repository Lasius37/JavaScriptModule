    Sélectionner des éléments

    Manipuler le DOM nécessite de sélectionner un (ou plusieurs) éléments HTML.

    • Sélectionner un élément (le premier qui correspond) : 
      const titre = document.querySelector("h1");
const zone = document.querySelector("#zone");
const bouton = document.querySelector(".btn");
    • Sélectionner plusieurs éléments : 
      const items = document.querySelectorAll(".item");

for (const element of items) {
    console.log(element);
}
    • Exemples de sélecteurs CSS utiles : 
      document.querySelector("header nav");
document.querySelector("ul li");
document.querySelector("input[type='text']");
document.querySelector("button.primary");
    • Si une sélection ne trouve rien, la valeur renvoyée est null.

    Modifier un élément

    Une fois un élément sélectionné, on peut modifier son texte, ses attributs, ses classes.

    • Modifier le texte : 
      const p = document.querySelector("#message");

p.textContent = "Bonjour";  // texte brut
p.innerHTML = "Bonjour <strong>à tous</strong>";  // contenu HTML interprétable par le navigateur
    • Attributs : 
      const img = document.querySelector("img");

img.setAttribute("alt", "Photo de profil");
const src = img.getAttribute("src");
    • Classes CSS (recommandé) : 
      const bloc = document.querySelector(".box");

bloc.classList.add("active");
bloc.classList.remove("hidden");
bloc.classList.toggle("selected");

    Créer des éléments

    On peut créer des éléments HTML en JavaScript, puis leur donner du contenu, des classes et des attributs.

    • Créer un élément : 
      const li = document.createElement("li");
li.textContent = "Nouvel élément";
    • Ajouter une classe et un attribut : 
      li.classList.add("item");
li.setAttribute("data-id", "12");

    À cette étape, les éléments existent en mémoire, mais ne sont pas encore dans la page. Pour les afficher, il faut les insérer dans le DOM.

    Insérer des éléments dans la page

    Pour insérer un élément dans la page, il faut d’abord sélectionner un parent, puis l'ajouter avec appendChild ou insertAdjacentElement .

    • Ajouter à la fin : 
      const ul = document.querySelector("#liste");
const li = document.createElement("li");
li.textContent = "Élément ajouté";

ul.appendChild(li);
    • Insérer à un endroit précis : 
      const p = document.querySelector("#message");
const span = document.createElement("span");
span.textContent = " (important)";

p.insertAdjacentElement("beforeend", span);

      Positions possibles : beforebegin, afterbegin, beforeend, afterend.

    • Remplacer un contenu existant (simple) : 
      const zone = document.querySelector("#zone");
zone.innerHTML = "<p>Nouveau contenu</p>";

      Attention : utilier ce paramètre remplace ce qui existe déjà.

    Naviguer dans le DOM

    Naviguer dans le DOM signifie se déplacer depuis un élément vers ses parents, ses enfants ou ses frères/sœurs.

    • Aller vers le parent : 
      const item = document.querySelector(".item");
const parent = item.parentElement;
    • Aller vers les enfants : 
      const ul = document.querySelector("ul");

console.log(ul.children);  // HTMLCollection
console.log(ul.firstElementChild);
console.log(ul.lastElementChild);
    • Frères / sœurs : 
      const first = ul.firstElementChild;

console.log(first.nextElementSibling);
console.log(first.previousElementSibling);
    • Chercher à l’intérieur d’un élément (scoper une sélection) : 
      const card = document.querySelector(".card");
const titre = card.querySelector("h3");

    Pour éviter de tomber sur des nœuds "texte", il est préférable d'utiliser les versions Element ( firstElementChild, etc.) plutôt que firstChild.

    Erreurs fréquentes

    • Oublier que querySelector peut renvoyer null : erreur au moment de modifier l’élément.
    • Confondre querySelector et querySelectorAll : problème de sélection.
    • Essayer d’utiliser une méthode d’élément sur une liste d'éléments : il faut parcourir le résultat de querySelectorAll.
    • Utiliser innerHTML pour tout : risque de remplacer trop de contenu.
    • Confondre childNodes et children : childNodes inclut les textes et retours à la ligne.
