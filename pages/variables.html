Déclarer une variable

Les variables sont des boîtes, avec une étiquette (leur nom), et qui contiennent une donnée. On les crée à l'aide du mot-clé let , et on les nomme en camelCase.

Dans les lignes ci-dessous, les textes en clair sont des commentaires.

  • Pour déclarer une variable sans lui affecter de valeur explicite : 
    let maVariable1;
// la variable a pour valeur undefined
  • Pour un nombre entier ou décimal : 
    let maVariable2 = 3;
// la variable a pour valeur le nombre entier 3
  • Pour une chaîne de caractères : 
    let maVariable3 = "mon prénom";
// la variable a pour valeur le texte "mon prénom"
  • Pour un booléen : 
    let maVariable4 = true;
// la variable a pour valeur true
  • Pour une valeur volontairement vide : 
    let maVariable5 = null;
// la variable a pour valeur null
  • Quand on veut rendre impossible une réassignation d'une variable, on utilise le mot-clé const. On nomme cette constante en SCREAMING_SNAKE_CASE. Par exemple: 
    const MA_CONSTANTE = 32;
    L'utilisation de const empêche la réassignation de la variable.

Appeler une variable

On utilise une variable créée plus haut.

  • On peut faire des calculs : 
    let somme = maVariable1 + maVariable2;
// calcul d'une somme, stockage dans une autre variable
  • On peut afficher un résultat : 
    console.log(somme);
// on affiche le résultat brut dans la console
  • On peut utiliser des chaînes de caractères formatées, ou template literals (ou template strings). Une template literal est une chaîne de caractères qui permet d'intégrer des variables ou des expressions. 
    let montant = 250;
console.log(`Votre compte est crédité de ${montant} €`);

Utiliser les principaux types de variables

Les variables ont un type donné. Chaque type est utilisable d'une façon qui lui est propre.

  • Pour connaître le type d'une variable, on peut taper la commande : 
    console.log(typeof maVariable);
// on affiche le type de la variable
  • Un nombre entier ou décimal est de type number.
  • Une chaîne de caractères est de type string.
  • Un booléen est de type boolean. Il peut prendre la valeur true ou false.
  • Il existe d'autres types de variables ; ici, on n'a présenté que les plus simples.

Erreurs fréquentes

  • Utilisation d'une variable non déclarée : ReferenceError.
  • Déclaration d'une constante sans affectation de valeur : SyntaxError.
  • Oubli des "" dans une chaîne de caractères : SyntaxError ou ReferenceError selon le cas.
  • Oublie des ` dans les template literals : SyntaxError ou comportement anormal.

Exercices

  1. Lire du code : qu'affiche le code suivant ?

    let monNom = "Toto";
console.log(`Je m'appelle ${monNom}.`);
    • message d'erreur
    • Je m'appelle ${monNom}.
    • Je m'appelle Toto.
    • Je m'appelle ${Toto}.
    • Je m'appelle monNom.

  2. Lire et écrire du code : corrigez les incohérences et respectez les conventions.

    let pi=3.14
console.log(`La valeur de {pi} est {3.14}`)

  3. Écrire du code : coder le programme suivant.

    • Créer une variable contenant votre prénom, et une autre contenant votre âge.
    • Afficher un message du type : "Bonjour Paul, vous avez 18 ans." en utilisant les template literals.
