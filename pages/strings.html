    Manipuler du texte

    Une chaîne de caractères (string) représente du texte. En JavaScript, une chaîne est immutable : on ne peut pas modifier directement ses caractères. On crée donc une nouvelle chaîne à partir de l’ancienne.

    • Concaténer deux chaînes : 
      const prenom = "Alice";
const phrase = "Bonjour " + prenom + " !";
console.log(phrase);
    • Utiliser un template string (recommandé) : 
      const age = 12;
const phrase = `Je m'appelle ${prenom} et j'ai ${age} ans.`;
console.log(phrase);
    • Accéder à un caractère : 
      const mot = "Python";
console.log(mot[0]);  // "P"
    • Longueur d’une chaîne : 
      console.log(mot.length);  // 6

    Pour manipuler du texte efficacement, on combine length, découpage, transformation, et recherche.

    Découper une chaîne

    Découper une chaîne sert à extraire une partie du texte : un mot, un préfixe, une extension de fichier, un morceau entre deux positions.

    • Extraire avec slice : 
      const mot = "javascript";

console.log(mot.slice(0, 4));  // "java"
console.log(mot.slice(4));     // "script"

      slice renvoie une nouvelle chaîne de caractères.

    • Utiliser des indices négatifs (à partir de la fin) : 
      const fichier = "notes.json";

console.log(fichier.slice(-4));  // "json"
    • Découper en morceaux avec split : 
      const phrase = "pomme,banane,orange";
const morceaux = phrase.split(",");

console.log(morceaux);  // ["pomme","banane","orange"]

      split renvoie un tableau.

    Transformer une chaîne

    Transformer une chaîne sert à normaliser du texte : enlever des espaces, uniformiser la casse, remplacer un morceau, formater une saisie utilisateur.

    • Mettre en majuscules / minuscules : 
      const mot = "JavaScript";

console.log(mot.toUpperCase());  // "JAVASCRIPT"
console.log(mot.toLowerCase());  // "javascript"
    • Supprimer les espaces au début/fin avec trim : 
      const saisie = "   Alice   ";
console.log(saisie.trim());  // "Alice"
    • Remplacer du texte : 
      const phrase = "Bonjour Paul";
console.log(phrase.replace("Paul", "Alice"));
    • Remplacer plusieurs occurrences : 
      const phrase = "a-a-a";
console.log(phrase.replaceAll("a", "b"));  // "b-b-b"

    Rechercher dans une chaîne

    Rechercher dans une chaîne sert à vérifier si un texte contient un mot, trouver une position, ou contrôler la présence d'un caractère par exemple.

    • Vérifier si un texte contient quelque chose : includes 
      const phrase = "Bienvenue sur le site";
console.log(phrase.includes("site"));  // true
    • Trouver la position : indexOf 
      const email = "alice@mail.com";
console.log(email.indexOf("@"));  // 5
    • Vérifier le début ou la fin : 
      const fichier = "notes.json";

console.log(fichier.startsWith("notes"));  // true
console.log(fichier.endsWith(".json"));  // true

    Erreurs fréquentes

    • Oublier que les chaînes sont immutables : les méthodes renvoient une nouvelle chaîne.
    • Confondre slice et split : slice renvoie une chaîne, split renvoie un tableau.
    • Comparer du texte sans normaliser (espaces, casse) : comparaisons fausses.
