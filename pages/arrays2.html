    Transformer un tableau

    map() crée un nouveau tableau en transformant chaque élément de l’original. Le tableau initial n’est pas modifié. 

    const notes = [8, 12, 15];

const doubles = notes.map((n) => n * 2);

console.log(doubles);

    Filtrer un tableau

    filter() crée un nouveau tableau contenant uniquement les éléments qui respectent une condition. 

    const notes = [8, 12, 15, 9];

const admis = notes.filter((n) => n >= 10);

console.log(admis);

    Calculer à partir d'un tableau

    reduce() permet de transformer un tableau en une seule valeur (somme, moyenne, total, etc.). 

    const notes = [10, 12, 15];

const somme = notes.reduce((acc, n) => acc + n, 0);

console.log(somme);

    Paramètres :

    • acc : accumulateur
    • n : élément courant ("boucle")
    • 0 : valeur initiale de acc

    Rechercher un élément

    find() renvoie le premier élément qui correspond à une condition. 

    const notes = [8, 12, 15];

const note = notes.find((n) => n >= 10);

console.log(note);

    Si aucun élément ne correspond, la méthode renvoie undefined.

    Copier et fusionner

    L’opérateur ... (spread) permet de copier ou fusionner des tableaux. 

    const a = [1, 2];
const b = [3, 4];

const copie = [...a];
const fusion = [...a, ...b];

console.log(fusion);

    On l'utilise pour éviter les effets de référence non désirés ou pour obtenir un tableau à partir de plusieurs.

    Extraire des valeurs

    Le destructuring permet d’extraire facilement des valeurs d’un tableau. 

    const notes = [12, 15, 9];

const [a, b] = notes;

console.log(a);
console.log(b);

    On peut ignorer une valeur : 

    const [premiere, , troisieme] = notes;

    Erreurs fréquentes

    • Oublier que map() et filter() ne modifient pas le tableau d’origine : risque de résultat inattendu.
    • Oublier la valeur initiale dans reduce() : résultat inattendu.
    • Confondre find() et filter() : comportement inattendu.
    • Copier un tableau avec = au lieu de ... : problème de référence.

    Exercices

    1. Lire du code : qu'affiche la console ?

      const tab = [2, 4, 6];

const result = tab
    .map((n) => n + 1)
    .filter((n) => n > 4);

console.log(result);

    2. Corriger le code

      const notes = [10, 12];
const copie = notes;

copie.push(15);

console.log(notes);
