    Énoncé

    Réaliser une calculatrice en JavaScript. Votre programme devra respecter le cahier des charges ci-dessous :

    1. Format des données

      • Les données sont stockées dans des variables.

    2. Fonctionnement

      • L'utilisateur saisit un nombre, puis un signe opératoire, puis un autre nombre.
      • L'ordinateur affiche le calcul et son résultat.
      • L'utilisateur peut décider de faire autant de calculs qu'il le veut avant de quitter.

    Conseils de réalisation

    • Stocker les saisies utilisateur dans des variables aux noms explicites.
    • Penser à convertir les nombres en number.
    • Coder une seule opération dans un premier temps (l'addition).
    • Utiliser une template literal pour afficher le résultat proprement.
    • Envisager les 4 possibilités d'opérations avec une structure conditionnelle.
    • Ajouter une boucle pour recommencer.

    Solution

    1. Stockage des saisies utilisateur dans des variables sous forme de nombres.
    2. Codage de l'addition et affichage du résultat soigné.
    3. Codage des autres opérations.
    4. Ajout d'une boucle.
