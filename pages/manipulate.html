    Faire des calculs

    JavaScript est un excellent calculateur. Il réalise de nombreuses opérations mathématiques avec les symboles suivants (les priorités sont les mêmes qu'en mathématiques) :

    addition + 5 + 2 7
    soustraction - 5 - 2 3
    multiplication * 5 * 2 10
    division / 5 / 2 2.5
    division (reste) % 5 % 2 1
    puissance ** 5 ** 2 25 
    console.log(5 + 2);
// JavaScript affiche 7

    Récupérer une saisie utilisateur

    Pour interagir avec l'utilisateur, il est nécessaire d'enregistrer les données qu'il saisit dans une variable. 

    let nomUtilisateur = prompt("Votre nom : ", "Toto");
// on peut proposer une valeur par défaut (ici, "Toto")

    Convertir le type d'une variable

    La fonction prompt() permet de récupérer une chaîne de caractères. Mais on peut avoir besoin de la convertir en nombre. 

    let maVariable1 = Number("37.5");  // renvoie le nombre 37.5
let maVariable2 = Boolean(1);  // renvoie le booléen true (pour 0, renvoie false)
let maVariable3 = String(32);  // renvoie la chaîne de caractères "32"
let maVariable4 = Boolean("0");  // renvoie le booléen true : chaîne non vide
let maVariable5 = Boolean("");  // renvoie le booléen false : chaîne vide
let maVariable6 = String(false);  // renvoie la chaîne de caractères "false"
let maVariable7 = Number(false);  // renvoie le nombre 0

    Utiliser les opérateurs selon le type de variables

    Il est fortement déconseillé de faire des calculs avec des variables de types différents : certains résultats sont surprenants et peuvent mener à des confusions. 

    console.log("3" + 5);  // renvoie "35" ; conversion en chaîne de caractères
console.log("3" - 5);  // renvoie -2 ; conversion en nombre

    Erreurs fréquentes

    • Calcul avec des variables de types différents (ex : "Coucou" - 3 ) : renvoie NaN (not a number).
    • Conversion d'une chaîne de caractères en nombre (ex: Number("abc")) : renvoie NaN (not a number).
    • Oubli du fait que prompt() renvoie une chaîne de caractères : renvoie NaN (not a number).
