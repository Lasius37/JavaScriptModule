    Créer une boucle for

    Cette boucle permet de répéter une série d'opérations un nombre déterminé de fois.

    for (let i = 1 ; i <= 5 ; i++) {  // pour chaque nombre entre 1 et 5
    console.log(i * 2);  // on affiche le double du nombre
    console.log(i ** 2);  // on affiche le carré du nombre
}

    Créer une boucle while

    Cette boucle permet de répéter une série d'opérations tant qu'une condition est vérifiée.

    let saisieUtilisateur;  // on déclare une variable
while (saisieUtilisateur !== "oui") {  // tant que la variable n'est pas égale à "oui"
    saisieUtilisateur = prompt("Tapez 'oui' en minuscules : ");  // on demande à l'utilisateur de taper "oui"
}

    Créer une boucle do...while

    Cette boucle ressemble à une boucle while, mais la condition est vérifiée après avoir exécuté le code une première fois.

    let saisieUtilisateur;  // on déclare une variable
do {  // on suit les instructions
    saisieUtilisateur = prompt("Tapez 'oui' en minuscules : ");
} while (saisieUtilisateur !== "oui");  // jusqu'à ce que la condition ne soit plus remplie

    Sortir d'une boucle

    Dans certains cas, une boucle while ou do...while peut être infinie. Pour en sortir, on utilise le mot break. On peut aussi l'employer dans une boucle for. 

    while (true) {  // boucle infinie : true est toujours vrai...
    let lettre = prompt("Taper une lettre : ");  // on demande une saisie à l'utilisateur
    if (lettre === "a") {  // on donne une condition
        console.log("OK !");  // on affiche un message de validation
        break;  // on sort de la boucle si la condition est vraie
    } else { // sinon...
        console.log("mauvais choix");  // on affiche un message d'erreur
    }

    Erreurs fréquentes

    • Oubli des {} autour du code de la boucle : répétition d'une seule instruction.
    • Oubli de mise à jour la variable de boucle (i++ ou i = i + 1) : boucle infinie.
    • Oubli du troisième élément dans une boucle for (i++) : SyntaxError.
    • Condition toujours vraie sans break : boucle infinie.
