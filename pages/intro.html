    Relier un fichier .js à un fichier .html.

    Le code JavaScript se place dans un fichier .js. On l'appelle dans le fichier .html. 

    <head>
    ...
    <script src="script.js" defer></script>
</head>
<!-- La balise <script> se place à la fin du <head> -->
<!-- defer permet de lancer le script à la fin du chargement du fichier html -->

    Lancer un programme en JavaScript

    Pour l'instant, on affiche le résultat du programme uniquement dans la console du navigateur. Pour cela,

    • On appelle la console dans le fichier .js. 
      console.log("Bonjour en JavaScript !");
    • Dans le navigateur, on inspecte la page (touche F12). Le contenu du message s'affiche dans la console. Attention, rien n'apparaît sur la page web.

    Erreurs fréquentes

    • Oubli de lien du fichier .js au fichier .html : pas d'affichage en console.
    • Erreur de chemin ou de nom du fichier .js : pas d'affichage en console.
    • Oubli de defer dans la balise <script> : lancement prématuré du programme.
