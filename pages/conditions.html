    Tester une condition

    Un test de condition renvoie un booléen (true ou false).

    • Cas n° 1 : "si..." 
      if (maVariable === 0) {  // si cette condition est vraie
    console.log("la variable est nulle");  // on affiche ce texte
}
    • Cas n° 2 : "si... sinon..." 
      if (maVariable > 0) {  // si cette condition est vraie
    console.log("la variable est positive");  // on affiche ce texte
} else {  // sinon
    console.log("la variable est négative ou nulle");  // on affiche ce texte
}
    • Cas n° 3 : "si... sinon si... sinon..." 
      if (maVariable < 0) {  // si cette condition est vraie
    console.log("la variable est négative");  // on affiche ce texte
} else if (maVariable > 0) {  // sinon si cette autre condition est vraie
    console.log("La variable est positive");  // on affiche ce texte
} else {  // sinon
    console.log("la variable est nulle");  // on affiche ce texte
}

    Utiliser les opérateurs de comparaison

    === ... est égal à ...
    !== ... est différent de ...
    < ... est strictement inférieur à ...
    > ... est strictement supérieur à ...
    <= ... est inférieur ou égal à ...
    >= ... est supérieur ou égal à ...

    Exemple :

    if (maValeur >= 0) {...
//si la valeur de la variable est positive ou nulle

    Tester plusieurs conditions : les opérateurs logiques

    && (and) ... et ... combine deux tests, les deux doivent être vrais
    || (or) ... ou ... combine deux tests, un seul doit être vrai
    ! (not) ... non ... inverse un test
    • Utilisation de && : 
      if (maValeur1 >= 0 && maValeur2 < 0) {...
// les deux conditions doivent être respectées
    • Utilisation de || : 
      if (maValeur3 === 0 || maValeur4 < 0) {...
// une seule des deux conditions doit être respectée
    • Utilisation de ! : 
      if (!utilisateurConnecte) {...
// ! inverse la réponse true / false

    Erreurs fréquentes

    • Confusion de = avec === : SyntaxError.
    • Oubli des () ou des {} après if, else if ou else : SyntaxError.
    • Condition impossible à atteindre : comportement anormal.
